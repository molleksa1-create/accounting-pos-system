# نظام المحاسبة والمخزون ونقطة البيع المتكاملة

نظام محاسبي متكامل مفتوح المصدر يجمع بين إدارة المشتريات والمخزون والتصنيع والمبيعات والتقارير المالية، مع دعم تطبيق كاشير منفصل للأندرويد.

## المميزات الرئيسية

### 1. إدارة المشتريات والموردين
- إنشاء وإدارة أوامر الشراء
- تتبع فواتير الموردين
- إدارة عمليات استقبال البضائع والجرد
- تتبع أرصدة الموردين

### 2. نظام المخزون المتقدم
- تتبع المنتجات والمواد الخام
- حركات المخزون (شراء، بيع، تحويل، تلف)
- جرد فعلي وتعديل المخزون
- تنبيهات المخزون المنخفض
- مواقع التخزين والمستودعات

### 3. نظام التصنيع والوصفات
- إنشاء وصفات المنتجات
- تحديد مكونات الوصفة
- أوامر الإنتاج والتخطيط
- تتبع المواد المستهلكة
- حساب تكاليف الإنتاج

### 4. نظام المبيعات والفواتير
- إنشاء أوامر البيع والفواتير
- إدارة العملاء والأرصدة
- دعم طرق الدفع المتعددة
- خصومات وضرائب مرنة
- تتبع حالة الفواتير

### 5. نقطة البيع (POS)
- واجهة كاشير سهلة الاستخدام
- جلسات الكاشير
- معاملات فورية
- تقارير يومية

### 6. التقارير والتحليلات
- تقارير المبيعات
- تقارير المشتريات
- تقارير المخزون
- التقارير المالية
- لوحة التحكم والمقاييس

### 7. إدارة المستخدمين والأدوار
- أدوار متعددة (مسؤول، مدير، محاسب، كاشير، أمين مخزن)
- صلاحيات مرنة
- سجل تدقيق شامل
- دعم الفروع المتعددة

## البنية التقنية

### المكدس التقني (Tech Stack)
- **الواجهة الخلفية**: Django 5.2 + Django REST Framework
- **قاعدة البيانات**: PostgreSQL
- **الواجهة الأمامية**: HTML/CSS/JavaScript + Django Templates
- **تطبيق الأندرويد**: React Native / Flutter
- **المصادقة**: Session-based Authentication

### البنية الهندسية
```
accounting_pos_system/
├── core/                    # التطبيق الأساسي (المستخدمون، الفروع، الموردين)
├── inventory/               # إدارة المخزون
├── manufacturing/           # نظام التصنيع والوصفات
├── accounting/              # المحاسبة والفواتير
├── pos/                     # نقطة البيع
├── reports/                 # التقارير والتحليلات
├── config/                  # إعدادات المشروع
├── templates/               # قوالب HTML
├── static/                  # ملفات ثابتة (CSS, JS, صور)
└── media/                   # ملفات المستخدمين (صور المنتجات)
```

## التثبيت والإعداد

### المتطلبات
- Python 3.11+
- PostgreSQL 12+
- pip

### خطوات التثبيت

1. **استنساخ المستودع**
```bash
git clone <repository-url>
cd accounting_pos_system
```

2. **إنشاء بيئة افتراضية**
```bash
python3.11 -m venv venv
source venv/bin/activate  # على Linux/Mac
# أو
venv\Scripts\activate  # على Windows
```

3. **تثبيت المكتبات**
```bash
pip install -r requirements.txt
```

4. **إعداد قاعدة البيانات**
```bash
python manage.py migrate
```

5. **إنشاء مستخدم إداري**
```bash
python manage.py createsuperuser
```

6. **تشغيل خادم التطوير**
```bash
python manage.py runserver
```

الآن يمكنك الوصول إلى التطبيق على: http://localhost:8000
